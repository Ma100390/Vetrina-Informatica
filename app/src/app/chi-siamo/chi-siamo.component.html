  • Java - Competenze nello sviluppo di applicazioni Java.
  • JavaScript - Esperienza nell'utilizzo di JavaScript per lo sviluppo di interfacce utente dinamiche.
  • TypeScript - Conoscenza avanzata di TypeScript per lo sviluppo di applicazioni robuste e scalabili.
  • .NET - Competenze nell'utilizzo della piattaforma .NET per lo sviluppo di applicazioni Windows e web.

Framework e Librerie

Competenze nell'utilizzo di vari framework e librerie per lo sviluppo di applicazioni web moderne.

  • Conoscenza approfondita di Bootstrap per la creazione di layout responsive e componenti stilizzati.
  • Utilizzo di Angular Material per la progettazione di interfacce utente coerenti e moderne.

Frontend Development

  • Sviluppo di pagine web dinamiche utilizzando HTML5, CSS3 e JavaScript.
  • Esperienza nell'utilizzo di React.js per la creazione di interfacce utente moderne.
  • Conoscenza di Next.js per lo sviluppo di applicazioni React.js avanzate.

  • Conoscenza di TypeScript per una programmazione più robusta e tipizzata.

Backend Development

Competenze nello sviluppo del backend con focus su Node.js, Express.js, MySQL e MongoDB.

  • Sviluppo di applicazioni server-side utilizzando Node.js.

DevOps
